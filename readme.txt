Description:
Matlab code to separate bathymetric data representing multiscale bedforms based on LOESS. Description of method in Zomer et al. (in press). 

example_application.m gives an example of how the code can be applied, based on two provided transects. 

References: 
Zomer, J.Y., S. Nashband, B. Vermeulen, A.J.F. Hoitink (in press), Rapidly migrating secondary bedforms can persist on the lee of slowly migrating primary river dunes, Journal of Geophysical Research: Earth Surface
